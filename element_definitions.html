Element Definitions Chapters

Chapters Two and Three

These tages are placed between < > for proper syntax

h1-h6
Tags used for identifying headings and subheadings in sequential size. H1 being largest to h6 being the smallest
p
Denotes a paragraph
b
This tag will make text enclosed within appear bold.
i
This tag will make text enclosed within appear italic.
sup
Text enclosed within will appear superscript. Examples include exponents. E2
sub
Text enclosed within will appear subscript. Examples include formula notation. NH4
br /
This tag will add a new line break within a line.
hr /
This tag will create a break between themes. Example, topics, scenes in a play.
strong
Used when content has a strong importance.
em
Used when content indicates emphasis that changes the meaning of sentence.
blockquote
Element used for quotes in excess of a paragraph.
q
Element used in short quotes within a paragraph.
abbr
Used for abbreviations or acronyms.
cite
Used to cite referenced work. Examples include literary works, research papers.
dfn
Used to deine the first instance of an academic concept or terminology.
address
Used for details of an author, physical address, phone number, or email.
ins
Used to show the insertion of text into a document.
del
Shows text meant for deletion.
s
Element used to show strikethrough of text.
ol
Denotes an ordered list.
li
When used with ol tag will generate numbers for items within list.
ul
Denotes an unordered list.
li
When used with ul tag, will generate bullet points beside items within list.
dl
Denotes a description list.
dt
Used within definition list. Defines a term to be defined.
dd
Used within definition list. Defines a definition of a term.
